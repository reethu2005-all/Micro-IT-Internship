# Currency Converter (GUI + Voice Interface)

A Python-based currency converter that supports real-time conversion between popular international currencies. It includes a graphical user interface (GUI) and a voice-based interface using speech recognition and text-to-speech.

## Features

- Real-time exchange rate conversion using ExchangeRate-API
- Voice input and output (speech recognition and text-to-speech)
- Simple and responsive GUI using Tkinter
- Currency code validation from a predefined dictionary
- Saves conversion history to a CSV file
- Handles invalid input and network errors

## Technologies Used

- Python 3.x
- requests
- pyttsx3
- speech_recognition
- tkinter
- csv
- datetime

## Supported Currencies

- USD - United States Dollar  
- EUR - Euro  
- INR - Indian Rupee  
- JPY - Japanese Yen  
- GBP - British Pound  
- AUD - Australian Dollar  
- CAD - Canadian Dollar  
- CHF - Swiss Franc  
- CNY - Chinese Yuan
